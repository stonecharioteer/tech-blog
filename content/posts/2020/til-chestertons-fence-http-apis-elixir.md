## TIL 2020-08-19

1. **[Chesterton's Fence](https://en.m.wikipedia.org/wiki/Wikipedia:Chesterton%27s_fence)** -
   Implementations might exist for a reason. Don't remove something until you
   understand why it was put there.

2. **[HTTP API for Cats](https://http.cat/)** - Fun way to learn HTTP status
   codes with cat images.

3. **[HTTP Status Code Cheatsheet](https://httpstatuses.com/)** - Comprehensive
   reference for HTTP status codes and their meanings.

4. **[Implementation of a Retro Finnish Game in Elixir](https://gitlab.com/Nicd/ex_speed_game/)** -
   Example of game development using Elixir.

5. **[Nerves: Add Agility to your IoT Development Cycle](https://www.nerves-project.org/)** -
   Platform for building IoT applications with Elixir.
